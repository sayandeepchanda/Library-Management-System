Library Management System -

This project is a desktop-based Library Management System developed as part of an academic innovation project.

The project started as a console-based Python program and was later converted into a GUI-based application using Tkinter with the help of AI tools for learning and guidance.

Features:
- User management
- Book management
- Issue and return tracking
- Search functionality
- Local database storage



How to Run (Source Code) -

Console Version (Base File – MySQL):

1. Install Python 3.x on the system and ensure it is added to PATH.
2. Install MySQL Server and make sure the MySQL service is running.
3. Open Command Prompt / Terminal and install MySQL connector for Python: pip install mysql-connector-python
4. Update MySQL connection details (host, username, password) inside the Python file if required.
5. Run the program using file:
   LIBRARY-MANAGEMENT-SYSTEM/Library Management System Base File


GUI Version (Advanced Version – SQLite):

1. Install Python 3.x on the system.
2. No external database setup is required as SQLite is used.
3. Run the program using file: LIBRARY-MANAGEMENT-SYSTEM/Library Management System GUI (Advanced)
